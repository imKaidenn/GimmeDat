## ✨ Features

- 🎬 **Videos** — YouTube, TikTok, Instagram, Pinterest, Twitter/X, Facebook, Reddit, SoundCloud, Vimeo, Twitch, Dailymotion, Bilibili — plus 1000+ more.
- 🖼 **Images** — paste any image link and grab the highest-quality version.
- 🎯 **Smart quality** — only the resolutions that actually exist light up, with size estimates. 4K → 240p + MP3.
- 📥 **Three ways in** — paste · drag & drop · ⚡ one-tap clipboard grab.
- 📁 **Your folder, flat** — pick where to save; videos drop straight in.
- 🎵 **HD merge + MP3** — bundled ffmpeg, no extra installs.
- 🐱 Pixel cyber-cat mascot · 🎶 looping 8-bit soundtrack · 🖥 resizable layout · ⌨ keyboard shortcuts.
- 💜 **Free forever.**
